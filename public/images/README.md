# Profile & student images

## Add Adeesha's photo

Save the uploaded portrait here as:

```
public/images/adeesha.jpg
```

That's it — the homepage hero, dashboard, and footer all reference
`/images/adeesha.jpg` automatically.

Until the file is added, the app shows an elegant gold monogram ("A")
in its place, so nothing looks broken.

## Student photos

You can drop student photos here too (e.g. `public/images/nethmi.jpg`)
and reference them from the student editor's **Photo URL** field, or use
any hosted image URL / Firebase Storage URL.

Recommended: portrait images around 1200×1500px, JPG or WebP.
